# **Detailed Summary – 42 Cybersecurity Pool**

## 1. Introduction
The 42 Cybersecurity Pool is an intensive program designed to immerse students in the practical fundamentals of information security. It covers networking, web security, cryptography, reverse engineering, and malware analysis through a series of progressive, hands-on projects.

---

## 2. Learning Objectives
- Understand threat models and common attack vectors.  
- Use essential pentesting and monitoring tools.  
- Analyze binaries and understand malicious behavior.  
- Implement basic cryptography techniques.  
- Strengthen Linux, scripting, debugging, and networking skills.  
- Develop autonomy and research skills.

---

## 3. Prerequisites & Work Environment
- Strong Linux command line proficiency.  
- Comfortable with shell scripting and system permissions.  
- Familiarity with Git and code editors.  
- Experience with virtual machines and sandbox environments.  
- Curiosity and the ability to learn independently.

---

## 4. Required Projects

### 4.1 Cybersecurity – arachnida (Web)
Introduction to **web pentesting**.  
Skills:
- Web application analysis.  
- Classic vulnerabilities: XSS, SQL/command injection, misconfigurations.  
- Enumeration and light brute-forcing.  
Objective: Think like an attacker and understand how web apps can be exploited.

---

### 4.2 Cybersecurity – ft_otp (OTP / Cryptography)
Implement a **One-Time Password system**.  
Skills:
- TOTP/HOTP standards.  
- HMAC, SHA-1, shared secrets.  
- Secure OTP generator implementation.  
- Time handling and Base32 encoding.  
Objective: Understand OTP mechanics and risks of poor implementation.

---

### 4.3 Cybersecurity – ft_onion (Web / Hardening)
Explore **hidden services and multi-layered architecture**.  
Skills:
- Reverse proxies, tunnels, multi-layer networking.  
- Analysis and hardening of layered environments.  
- Privacy and security across multiple web layers.  
Objective: Handle and secure a compartmentalized web architecture.

---

### 4.4 Cybersecurity – Reverse me (Reverse Engineering)
Introduction to **reverse engineering**.  
Skills:
- Binary analysis with tools (objdump, gdb, Ghidra, radare2).  
- Understanding low-level assembly.  
- Detecting simple protections (XOR, logic checks).  
Objective: Learn to deconstruct executables and uncover hidden behavior.

---

### 4.5 Cybersecurity – Stockholm (Malware)
Study and analyze a **simple malware**.  
Skills:
- Observe malware behavior.  
- Basic techniques: file encryption, local propagation, persistence.  
- Static and dynamic analysis.  
- Implement safe containment (sandboxing, logging).  
Objective: Understand malware behavior and safe analysis techniques.

---

### 4.6 Cybersecurity – Inquisitor (Network)
Focus on **network monitoring and analysis**.  
Skills:
- Packet capture and analysis (Wireshark, tcpdump).  
- Scanning and network reconnaissance.  
- Anomaly detection and data exfiltration.  
- Understanding protocol weaknesses.  
Objective: Develop network analyst skills and SOC mindset.

---

### 4.7 Cybersecurity – Vaccine (Web)
Advanced project on **patching and securing web applications**.  
Skills:
- Identify vulnerabilities in existing applications.  
- Properly patch security issues.  
- Write secure web code (input validation, hashing, sessions, auth).  
- Test and validate security improvements.  
Objective: Learn how to harden applications effectively.
